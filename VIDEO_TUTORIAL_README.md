# Video Tutorial Setup

## Database Setup

Jalankan SQL berikut di Supabase SQL Editor untuk membuat tabel `videos`:

```sql
create table if not exists public.videos (
  id bigint generated by default as identity primary key,
  title text not null,
  description text not null,
  video_url text not null,
  created_at timestamptz default now()
);

-- Enable RLS
alter table if exists public.videos enable row level security;

-- Public read access for videos
create policy if not exists "Public can select videos" on public.videos
  for select
  using (true);

-- Admin full access for videos
create policy if not exists "Admin can manage videos" on public.videos
  for all
  using (auth.email() = 'admin@company.com')
  with check (auth.email() = 'admin@company.com');
```

## Fitur Video Tutorial

### Admin Features
- **Kelola Video**: Tambah, edit, hapus video tutorial
- **Form Video**: Input judul, deskripsi, dan link Google Drive
- **Search**: Cari video berdasarkan judul atau deskripsi

### Public Features
- **Video List**: Tampilan grid semua video tutorial
- **Video Detail**: Halaman detail dengan embedded video
- **Google Drive Integration**: Auto-convert link Drive ke format embed

### File Structure
```
components/Admin/
├── VideoDashboard.tsx    # Dashboard admin video
└── VideoForm.tsx         # Form tambah/edit video

app/videos/
├── page.tsx              # List video tutorial
└── [id]/
    └── page.tsx          # Detail video tutorial

supabase/
└── videos.sql            # SQL schema untuk tabel videos
```

### Video URL Format
Link Google Drive harus dalam format:
```
https://drive.google.com/file/d/[FILE_ID]/view
```

Akan otomatis dikonversi ke format embed untuk ditampilkan di halaman detail.
